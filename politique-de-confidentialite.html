Données collectées via le formulaire de contact

Lorsque vous utilisez le formulaire de contact présent sur le site, les données suivantes sont collectées :

  • Nom : si vous le renseignez
  • Email : adresse email (obligatoire)
  • Téléphone : si vous le renseignez (optionnel)
  • Sujet : objet de votre demande
  • Message : contenu de votre message

Finalité

Les données collectées via le formulaire de contact sont utilisées pour :

  • Répondre aux demandes envoyées via le formulaire (prise de contact, informations, devis, rendez-vous)
  • Traiter vos demandes et vous contacter en retour

Base légale

Le traitement de vos données personnelles est fondé sur :

  • Intérêt légitime : répondre aux demandes entrantes via le formulaire de contact
  • Mesures précontractuelles : dans le cadre de demandes de devis ou de rendez-vous

Destinataires et sous-traitants

Destinataire : Vélos BIBI uniquement

Sous-traitant : Les messages sont transmis via le service Formspree, utilisé comme prestataire technique de formulaire. Formspree agit en tant que sous-traitant pour le traitement des données du formulaire de contact.

Les données transmises à Formspree sont utilisées uniquement pour l'envoi des messages et ne sont pas utilisées à d'autres fins.

Durée de conservation

Les données collectées via le formulaire de contact sont conservées jusqu'à 12 mois maximum, ou le temps nécessaire au traitement de votre demande, puis supprimées.

Droits des personnes

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : vous avez le droit d'obtenir une copie des données personnelles que nous détenons vous concernant
  • Droit de rectification : vous avez le droit de demander la correction de données inexactes ou incomplètes
  • Droit de suppression : vous avez le droit de demander la suppression de vos données personnelles
  • Droit d'opposition : vous avez le droit de vous opposer au traitement de vos données personnelles
  • Droit de limitation : vous avez le droit de demander la limitation du traitement de vos données
  • Droit à la portabilité : vous avez le droit de recevoir vos données dans un format structuré et couramment utilisé

Exercer ses droits / Contact RGPD

Pour exercer vos droits ou pour toute demande concernant vos données personnelles, vous pouvez nous contacter à l'adresse suivante :

Email : velosbibi@gmail.com

Nous nous engageons à répondre à votre demande dans un délai d'un mois maximum.

Cookies et stockage local

Aucun cookie publicitaire, aucun traceur marketing, aucune mesure d'audience tiers n'est utilisé sur ce site.

Le seul stockage utilisé sert à mémoriser votre choix de consentement aux cookies via le stockage local (localStorage) avec la clé vb_cookie_consent.

Pour plus d'informations détaillées sur les cookies, consultez notre page dédiée aux cookies.

Modifications de cette politique

Cette politique de confidentialité peut être modifiée à tout moment. Toute modification sera publiée sur cette page. Nous vous encourageons à consulter régulièrement cette page pour prendre connaissance des éventuelles mises à jour.
